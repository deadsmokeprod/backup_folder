# Assets

- `fonts/` — сюда положить `.ttf`/`.otf` шрифтов (Inter, Manrope, опционально M PLUS Rounded 1c).
  Приложение автоматически подхватит всё при старте через `theme/fonts.py`.
  Если папка пуста — используются системные шрифты (Segoe UI).

- `icons/` — SVG/PNG иконки в общем стиле. Сейчас не обязательно — приложение работает и без них.

- `bg/` — фоновые аниме-иллюстрации и пусто-состояния. На первое время используется одноцветный тёмный фон;
  позже можно положить сюда свой арт и подключить через QSS `background-image`.

- `app.ico` — иконка приложения, подтягивается в PyInstaller spec (`build/gui.spec`).
